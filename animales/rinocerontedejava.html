Rinoceronte de Java

El rinoceronte de Java vietnamita (Rhinoceros sondaicus annamiticus) se extinguió en Vietnam en 2010. El último ejemplar fue encontrado muerto de un disparo en el Parque Nacional Cat Tien.

Amenazas

El rinoceronte de Java se enfrenta en Vietnam a amenazas como la caza furtiva y la pérdida de hábitat.
